## 🔄 User Experience Flow

### During Contract Creation:

1. User selects **Agreement Type 4** or **5**
2. System displays all 5 សមិទ្ធកម្ម
3. For each សមិទ្ធកម្ម:
   - Show the title and timeline
   - Display 3 radio button options
   - User MUST select ONE option
4. User completes other contract fields
5. User signs and submits
6. System saves selections to `contract_deliverable_selections`

### During Contract Viewing/Printing:

1. Load contract data
2. Load selected options for each សមិទ្ធកម្ម
3. Display table with:
   - ល.រ (Number)
   - សមិទ្ធកម្ម (Deliverable title)
   - សូចនាករ (ONLY show the selected option, not all 3)
   - ពេលវេលាអនុវត្ត (Timeline)
